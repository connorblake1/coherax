coherax
=======

Coherent-basis optimization toolkit for bosonic quantum error-correcting codes.

Built on `JAX <https://github.com/jax-ml/jax>`_ and
`dynamiqs <https://github.com/dynamiqs/dynamiqs>`_, coherax provides
analytic closed-form fidelity formulas, CD+R circuit optimization,
transpose channel recovery, and coherent information benchmarks for
GKP, cat, binomial, and floating-basis codes.
